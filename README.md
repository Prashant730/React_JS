# React_JS

## Project Overview

This repository contains React projects and basic JavaScript files for learning and experimenting with React and modern web development tools.

### Main Contents

- **app.js, app1.js, script.js**: Standalone JavaScript files for practice and demos.
- **index.html**: Main HTML file for basic testing.
- **basicOfReact/**: Contains a React project scaffolded with Vite.

## Getting Started (basicOfReact)

1. Navigate to the `basicOfReact` folder:
   ```sh
   cd basicOfReact
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
4. Open your browser to the local server URL (usually `http://localhost:5173`).
