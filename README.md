# React_JS

## Project Overview

This repository contains multiple React projects and JavaScript files for learning, experimenting, and demonstrating modern web development concepts. Each subfolder is a self-contained project, most using React with Vite, and includes example components, assets, and configuration for rapid development.

---

## Main Contents

- **app.js, app1.js, script.js**: Standalone JavaScript files for practice and demos.
- **index.html**: Main HTML file for basic testing.
- **basicOfReact/**: Basic React + Vite project.
- **card_prop_project/**: React project demonstrating card components and props.
- **components/**: React project focused on reusable components (e.g., Card, Navbar).
- **props/**: React project for learning about props and component composition.
- **use_function_react/**: React + Tailwind CSS project for functional components and styling.

## Getting Started

Each subproject is a standalone React app (with Vite). To run any project:

1. Navigate to the project folder (e.g., `cd basicOfReact`)
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
4. Open your browser to the local server URL (usually `http://localhost:5173`)
